# Stop Loss คืออะไร? วิธีตั้ง Stop Loss ไม่ให้ขาดทุนหนัก

ถ้ามีสิ่งเดียวที่นักลงทุนมือใหม่ทุกคนควรรู้ก่อนเริ่มเทรด นั่นคือ **Stop Loss** เพราะตลาดหุ้นไม่มีการรับประกันว่าราคาจะไปในทิศทางที่คุณคาดไว้เสมอ การไม่มี Stop Loss เปรียบเหมือนขับรถโดยไม่คาดเข็มขัดนิรภัย

---

## Stop Loss คืออะไร?

**Stop Loss** คือคำสั่งที่ตั้งไว้ล่วงหน้าให้ระบบ **ขายหุ้นโดยอัตโนมัติ** เมื่อราคาลดลงถึงระดับที่กำหนด เพื่อจำกัดการขาดทุนไม่ให้เกินกว่าที่ยอมรับได้

### ตัวอย่างง่าย ๆ

- ซื้อหุ้น NVDA ที่ $500
- ตั้ง Stop Loss ที่ $475 (ขาดทุนสูงสุด $25 หรือ 5%)
- ถ้าราคาลงมาถึง $475 ระบบจะขายทันที ป้องกันไม่ให้ขาดทุนต่อเนื่อง

---

## ทำไม Stop Loss ถึงสำคัญมาก?

### ปัญหาของการถือหุ้นขาดทุนโดยไม่มี Stop Loss

| ขาดทุน | ต้องกำไรเท่าไหร่ถึงคืนทุน |
|--------|--------------------------|
| 10% | 11.1% |
| 20% | 25% |
| 33% | 50% |
| 50% | **100%** |
| 75% | **300%** |

เห็นไหมว่า ยิ่งขาดทุนมาก การฟื้นตัวยิ่งยากขึ้นแบบเอ็กซ์โปเนนเชียล หุ้นที่ลงมา 50% ต้องขึ้น 100% ถึงจะคืนทุน

---

## ประเภทของ Stop Loss

### 1. Fixed Stop Loss (Stop Loss แบบคงที่)

ตั้งที่ราคาหรือ % ที่แน่นอน เช่น:
- ซื้อที่ $100 → Stop Loss ที่ $95 (ขาดทุนสูงสุด 5%)

**ข้อดี:** เรียบง่าย คำนวณง่าย
**ข้อเสีย:** ไม่คำนึงถึงความผันผวนของหุ้นแต่ละตัว

### 2. ATR-based Stop Loss (ตาม Average True Range)

ใช้ความผันผวนเฉลี่ยของหุ้นในการกำหนด Stop Loss

สูตร: `Stop Loss = ราคาเข้า - (ATR × ตัวคูณ)`

ตัวอย่าง: ATR(14) = $8, ตัวคูณ = 2
- Stop Loss = $100 - ($8 × 2) = $84

**ข้อดี:** เหมาะกับความผันผวนของหุ้นแต่ละตัว
**ข้อเสีย:** ต้องคำนวณ ATR ก่อน

### 3. Technical Stop Loss (ตาม Chart)

วาง Stop Loss ต่ำกว่าจุดสำคัญทาง Technical:
- ต่ำกว่า Support Level
- ต่ำกว่า EMA 20 หรือ EMA 50
- ต่ำกว่า Low ของ Pattern

**ข้อดี:** สอดคล้องกับโครงสร้างตลาด ไม่โดน Stop ง่าย ๆ
**ข้อเสีย:** ต้องอ่านกราฟเป็น

### 4. Trailing Stop Loss (Stop Loss แบบเดินตาม)

Stop Loss ขยับขึ้นตามราคาอัตโนมัติ แต่ไม่ขยับลง

ตัวอย่าง: ซื้อที่ $100, Trailing Stop 5%
- ราคาขึ้นไป $110 → Stop Loss ขยับเป็น $104.50
- ราคาขึ้นไป $120 → Stop Loss ขยับเป็น $114
- ราคาลงมาถึง $114 → ขายทันที (ยังได้กำไร 14%)

**ข้อดี:** Lock in profit ได้อัตโนมัติ
**ข้อเสีย:** อาจโดน Stop ในการ Pullback ปกติ

---

## วิธีตั้ง Stop Loss ที่ถูกต้อง

### หลักการสำคัญ: วางหลัง Pattern ไม่ใช่ตามใจอยาก

Stop Loss ที่ดีควรอยู่ **"หลังโครงสร้างตลาด"** ที่ถ้าราคาลงมาถึงนั้นแสดงว่า Setup ของเราผิดแล้ว

**ตัวอย่างการวาง Stop Loss ที่ดี:**

1. **ซื้อที่ Breakout** → Stop Loss ต่ำกว่าจุด Breakout 1-2%
2. **ซื้อที่ Support** → Stop Loss ต่ำกว่า Support 1-2%
3. **ซื้อที่ EMA 20 Pullback** → Stop Loss ต่ำกว่า Low ของแท่งเทียนที่เด้ง

### ข้อผิดพลาดการตั้ง Stop Loss ที่ควรหลีกเลี่ยง

- **Stop Loss แน่นเกินไป** — โดน Stop จากการแกว่งปกติของหุ้น
- **Stop Loss หลวมเกินไป** — ขาดทุนมากเกินความจำเป็นก่อนออก
- **ไม่ตั้ง Stop Loss เลย** — อันตรายมากที่สุด
- **ขยับ Stop Loss ออกเมื่อราคาใกล้** — ทำลายวินัยทั้งระบบ

## Stop Loss กับ Position Sizing

Stop Loss ต้องคู่กับ Position Sizing เสมอ เพื่อควบคุมความเสี่ยงต่อ Trade

### สูตรคำนวณขนาด Position

```
จำนวนหุ้น = (มูลค่าพอร์ต × %Risk สูงสุดต่อ Trade) ÷ (ราคาเข้า - ราคา Stop Loss)
```

**ตัวอย่าง:**
- พอร์ต: 500,000 บาท
- Risk ต่อ Trade: 1% = 5,000 บาท
- ราคาเข้า: 50 บาท
- Stop Loss: 47 บาท (ห่าง 3 บาท)
- จำนวนหุ้ม = 5,000 ÷ 3 = **1,666 หุ้น** (~83,300 บาท)

ผลคือ ถ้า Stop โดน ขาดทุน 1,666 × 3 = 4,998 บาท ≈ 1% ของพอร์ต

## Mental Stop Loss vs. Hard Stop Loss

**Hard Stop Loss:** ตั้งคำสั่ง Stop ในระบบของ Broker จริง ๆ
- ข้อดี: ไม่ต้องเฝ้าหน้าจอ, ปฏิบัติโดยอัตโนมัติ
- ข้อเสีย: อาจโดน Stop Hunting จาก Market Maker

**Mental Stop Loss:** จดไว้ในใจหรือในโน้ต แล้วขายด้วยตัวเองเมื่อถึงระดับ
- ข้อดี: ยืดหยุ่นกว่า ไม่โดน Stop Hunting
- ข้อเสีย: ต้องมีวินัยสูง มักไม่ยอมขายเมื่อถึงเวลาจริง

**คำแนะนำ:** สำหรับมือใหม่ ใช้ Hard Stop Loss ดีกว่าเสมอ

---

## สรุป

Stop Loss ไม่ใช่สัญญาณของความล้มเหลว แต่คือ **เครื่องมือที่นักเทรดมืออาชีพใช้ทุกคน** การตั้ง Stop Loss ที่ดีต้องอิงกับโครงสร้างตลาด คำนวณ Position Sizing ให้เหมาะสม และที่สำคัญที่สุดคือ **ปฏิบัติตามมันอย่างมีวินัย** ไม่ว่าจะรู้สึกยังไง

> ⚠️ บทความนี้เป็นเพียงข้อมูลเพื่อการศึกษา ไม่ใช่คำแนะนำทางการเงิน ควรศึกษาเพิ่มเติมและปรึกษาผู้เชี่ยวชาญก่อนตัดสินใจลงทุน
