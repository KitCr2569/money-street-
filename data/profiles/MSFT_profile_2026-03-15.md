## 🏢 MSFT คืออะไร?

Microsoft Corporation เป็นบริษัทซอฟต์แวร์และ cloud computing รายใหญ่ที่สุดของโลก ดำเนินธุรกิจผ่าน 3 ส่วนหลัก: **Productivity & Business Processes** (Microsoft 365, LinkedIn, Dynamics 365), **Intelligent Cloud** (Azure, GitHub, Nuance) และ **Personal Computing** (Windows, Xbox, Bing/Copilot, Surface) Microsoft เป็นผู้นำในตลาด enterprise software และ cloud infrastructure โดย Azure เป็นผู้ให้บริการ cloud อันดับ 2 รองจาก AWS

ก่อตั้งปี 1975 โดย Bill Gates และ Paul Allen สำนักงานใหญ่อยู่ที่ Redmond, Washington มีพนักงานประมาณ 228,000 คน

---

## 💡 จุดเด่น

- **Revenue เติบโต 16.7%** — Azure cloud + AI Copilot ขับเคลื่อนการเติบโต แม้บริษัทจะใหญ่มากแล้ว
- **Profit Margin 39%** — กำไรสุทธิเกือบ 40 เซ็นต์ต่อทุก 1 ดอลลาร์รายได้ แสดงถึง operating leverage ระดับสูง
- **Gross Margin 68.6%** — สูงลิ่ว สะท้อนธุรกิจซอฟต์แวร์ + cloud ที่มี cost structure ดีเยี่ยม
- **Dividend Yield 0.92%** — จ่ายปันผลสม่ำเสมอ + buyback เพิ่ม shareholder return
- **53 นักวิเคราะห์ให้ Strong Buy** — Target เฉลี่ย $594.62 สูงกว่าราคาปัจจุบัน 50%!
- **Copilot AI** — เป็นผู้นำการ monetize AI ในระดับ enterprise ผ่าน Microsoft 365 Copilot
