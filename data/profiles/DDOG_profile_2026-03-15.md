## 🏢 DDOG คืออะไร?

Datadog, Inc. เป็นบริษัทชั้นนำด้านแพลตฟอร์ม Observability และ Security สำหรับแอปพลิเคชันบน Cloud ก่อตั้งในปี 2010 มีสำนักงานใหญ่ที่นิวยอร์ก บริษัทให้บริการในสหรัฐอเมริกาและทั่วโลก มีพนักงาน 8,100 คน

แพลตฟอร์ม Datadog รวมเครื่องมือหลากหลายในที่เดียว ทั้งการติดตาม Infrastructure, Application Performance Monitoring (APM), Log Management, Security, Real User Monitoring, LLM Observability, Cloud Cost Management, Network Monitoring และระบบตอบสนองเหตุการณ์ผ่าน AI
