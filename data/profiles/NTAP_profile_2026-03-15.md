## 🏢 NTAP คืออะไร?

NetApp, Inc. เป็นบริษัทชั้นนำด้านโซลูชันการจัดการข้อมูลสำหรับองค์กร (Enterprise Data Management) ก่อตั้งเมื่อปี 1992 และมีสำนักงานใหญ่ที่ San Jose, California บริษัทให้บริการซอฟต์แวร์ ระบบ และบริการสำหรับการบริหารโครงสร้างพื้นฐานข้อมูลทั้งในองค์กร (On-premises) และบน Multi-cloud ผ่าน 2 กลุ่มธุรกิจหลัก ได้แก่ Hybrid Cloud ซึ่งนำเสนอ All-Flash Storage, ONTAP OS และระบบ SAN และ Public Cloud ซึ่งให้บริการ Cloud Storage ผ่าน Azure NetApp Files, Amazon FSx for NetApp ONTAP และ Google Cloud NetApp Volumes

NetApp ให้บริการลูกค้าในอุตสาหกรรมพลังงาน การเงิน รัฐบาล เทคโนโลยี Life Sciences สุขภาพ และโทรคมนาคม มีพนักงาน 11,700 คน
